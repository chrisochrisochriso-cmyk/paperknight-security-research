Overview

This repository demonstrates the capabilities of the PaperKnight scanner, one of the fastest domain and network telemetry tools available. It includes anonymized, hashed, and redacted data from high-speed scans of public infrastructure.

All data is legally compliant, with no sensitive personal information included. It’s ideal for research, analytics, commercial intelligence, or integration into data pipelines.

What’s Included

Sample Dataset: ~100 lines of anonymized telemetry data per domain, with hashed identifiers.

Scan Metadata: Timing, scan rate, and exposed service types (e.g., SSH detection) for benchmarking.

README Pricing & Contact: Guidance for acquiring the full dataset.

Showcase: Examples of the PaperKnight scanner in action, highlighting speed, accuracy, and scalability.
